Dotfiles
========

My dotfiles. Configured the way I want them to be.
Clean and purely beautiful.

## Installation

```bash
$ git clone https://github.com/hpurmann/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
$ ./install
```

## Brew Packages

First get [Homebrew](http://brew.sh/).

```bash
$ brew tap Homebrew/bundle
$ brew bundle
```
