Dotfiles
========

My dotfiles. Configured the way I want them to be.

## Installation

```
$ git clone https://github.com/hpurmann/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
$ ./makesymlinks.sh
```

## YouCompleteMe

You have to compile YouCompleteMe after installing.

```
$ cd ~/dotfiles/vim/bundle/YouCompleteMe/
$ ./install.sh
```
